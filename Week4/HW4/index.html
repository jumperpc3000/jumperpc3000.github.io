Basics Alignment Sizing Notes

Core idea

Flexbox Is For One Main Direction At A Time.

A flex container controls how its direct children line up, space out, grow, shrink, wrap, and align. The parent decides the layout rules. The children react to those rules.

container

Display: flex; turns the parent into a flex container.

container and items

1. Parent Vs Child

The container gets flex properties. The items inside it become flex items. 

.container {
  display: flex;
}
flex-direction

2. Main Axis Direction

The main axis can go left to right, right to left, top to bottom, or bottom to top.

row
column 
.container {
  flex-direction: row;
}
flex-wrap

3. Wrapping

By default, flex items try to stay on one line. Wrap lets them move to a new line. 

.container {
  flex-wrap: wrap;
}
justify-content

4. Main-Axis Alignment

Use justify-content to control how items spread out across the main axis.

Center
Space-Between
Space-Evenly 
.container {
  justify-content: space-evenly;
}
align-items

5. Cross-Axis Alignment

Align-items changes how items sit across the cross axis of the current line.

center
stretch 
.container {
  align-items: center;
}
align-content

6. Multiple Lines

Align-content only matters when the container has extra cross-axis space and multiple lines. 

.container {
  flex-wrap: wrap;
  align-content: space-around;
}
order

7. Visual Order

Order changes display order, but it does not change the source order in the HTML.

grow, shrink, basis

8. Item Sizing Rules

Flex-grow shares extra space. Flex-shrink helps avoid overflow. Flex-basis sets the starting size.

1
2
1

.item {
  flex-grow: 1;
  flex-shrink: 1;
  flex-basis: 50px;
}
align-self

9. One Item Can Break The Rule

Align-self overrides the shared cross-axis alignment for a single flex item. 

.item-special {
  align-self: flex-end;
}
quick notes

10. When To Use Flexbox

  • Use flexbox for rows, columns, nav bars, and cards.
  • Use grid when you know both rows and columns ahead of time.
  • Flexbox is excellent for components inside a larger layout.
  • Gap works in flexbox and gives clean spacing between items.
