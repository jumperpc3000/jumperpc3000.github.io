Study Break Control Center

This website helps students plan a useful study break. It uses the five JavaScript interactions, listed below, with event listeners, events, and functions.

  • When the webpage loads, the greeting message changes based on the time of day.
  • When the user clicks the “Press me!” button, an alert appears that says “Pressed.”
  • When the user clicks inside the reminder text box, the focus message changes.
  • When the user clicks the “Add Reminder” button, the typed reminder is added to the list.
  • When the user hovers over the break idea section, a new break idea appears.
